# WorkerOS

An operating system that boots inside a Web Worker and runs JavaScript or
WebAssembly programs as **real processes** — spawned, streamed, killable, and
visible in `ps`.

The executable format is a JS or WASM module instead of a native binary, and the
"CPU" is the host's own JS/WASM engine. The kernel is written in Rust (compiled to
WASM) and owns the VFS, the process table, module resolution, and capabilities.
